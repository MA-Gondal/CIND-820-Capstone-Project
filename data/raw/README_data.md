# Data Access Note — USAID SCMS Delivery History Dataset

## Dataset Details

| Attribute | Detail |
|---|---|
| Full Name | USAID SCMS Delivery History Dataset |
| Kaggle URL | https://www.kaggle.com/datasets/princehobby/supply-chain-shipment-dataset |
| Original Source | USAID / Supply Chain Management System (SCMS) Program |
| License | CC-BY (USAID original) |
| Records | 10,324 rows, 33 columns |
| Coverage | 2006–2015 |

## Download Instructions

1. Go to the Kaggle URL above
2. Download the CSV file
3. Place it in this folder (`data/raw/`) with the filename:
   `Suppy_Chain_Shipment_Data.csv`

**Note:** The filename contains a typo ("Suppy" instead of "Supply") — 
this is the original Kaggle uploader's filename. Keep it as-is to match 
the file path referenced in the notebooks.

## Note on Raw Data

The CSV file (`Suppy_Chain_Shipment_Data.csv`) is committed to this repository
for reproducibility purposes. If the file is unavailable, it can also be
downloaded directly from the Kaggle URL above.
